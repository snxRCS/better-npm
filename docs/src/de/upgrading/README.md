# Aktualisierung

```bash
docker compose pull
docker compose up -d
```

Dieses Projekt wird automatisch alle Datenbanken oder andere Anforderungen aktualisiert, sodass Sie nicht irgendwelchen verrückten Anweisungen folgen müssen. Diese Schritte oben laden die neuesten Updates herunter und erstellen die Docker-Container neu.

Siehe die [Release-Liste](https://github.com/snxRCS/better-npm/releases) für versionsspezifische Upgrade-Schritte.
